## ✨ Features

- **Dual-layout architecture** — Fully separate desktop and mobile experiences
- **Barba.js transitions** — Smooth PJAX page transitions
- **GSAP animations** — Premium scroll-triggered animations
- **Lenis smooth scroll** — Buttery scroll experience on desktop
- **Mobile-first** — Dedicated light-mode mobile site with hamburger nav
- **Legal pages** — Privacy Policy, Terms of Service, Cookie Policy
- **Form ready** — Inquiry forms with null API (ready to connect any backend)

## 🚀 Deploy

This site deploys automatically via **GitHub Actions** to GitHub Pages on every push to `main`.

To enable GitHub Pages:
1. Go to **Settings → Pages**
2. Set Source to **GitHub Actions**
3. Push to `main` — the workflow handles the rest
