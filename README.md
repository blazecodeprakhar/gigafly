# Giga Fly Replica

A high-performance, premium frontend replica of [Giga Fly](https://gigafly.com/).

## 🚀 Features
- **GSAP Animations**: Smooth reveal effects and magnetic buttons.
- **Barba.js**: Seamless page transitions.
- **Lenis Scroll**: High-end inertial scrolling.
- **Localized Assets**: All images, scripts, and styles hosted locally.

## 🌐 Deployment
This project is optimized for **Netlify**. 
1. Push to GitHub.
2. Connect the repository to Netlify.
3. No build command required (Site is static).
